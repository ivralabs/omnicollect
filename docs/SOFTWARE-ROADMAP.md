# OmniCollect Software Roadmap

*Locked 2026-06-23*

## Phase 1 — MVP (Months 1–6)

Core measurement. Prove the concept. Get pilot sites live.

- Vehicle counting (real-time, site-specific)
- Vehicle make & model detection (YOLOv8 + SA fine-tune)
- Vehicle colour detection
- Number plate hashing (on-device, POPIA compliant)
- People counting + dwell time
- Basic analytics dashboard (site map, hourly counts, vehicle class breakdown)
- Anomaly detection + alerts (traffic spike/drop, camera offline, unusual activity)
- Post-campaign basic report (impressions, peak times, vehicle class)

## Phase 2 — Intelligence (Months 7–12)

Layer context and depth onto the raw data.

- **Weather correlation engine** — OpenWeatherMap overlay, traffic vs conditions analysis
- **Dwell time measurement** — per vehicle class, per time of day
- **Seasonal audience profiling** — 12-month pattern per site, school holidays, public holidays
- **Event impact tracker** — automatic spike detection, match days, concerts
- **Full post-campaign report** — PDF + Excel + API, verified numbers, auto-generated in 30 seconds
- **Queue detection** — vehicles stopped at lights = captive audience flag

## Phase 3 — Network (Year 2)

The features that separate OmniCollect from every global competitor.

- **Cross-site audience journey** — hashed plate tracking across sites = verified unique reach + frequency capping for OOH. Nobody has built this globally.
- **OmniCollect Score** — single 1–100 composite per site (impressions, vehicle class quality, dwell, consistency, anomaly rate). Industry benchmark.
- **Creative performance scoring** — marry CMS playout log to audience data = A/B testing for physical billboards.
- **Agency planning dashboard** — audience data across all verified sites for campaign planning
- **Competitor activity monitor** — share of attention between nearby sites

## Phase 4 — Platform (Year 3+)

The Bloomberg terminal play.

- **Programmatic OOH API** — license real-time audience data to Hivestack, Vistar, Broadsign for automated buying
- **Predictive forecasting** — AI model trained on 2–3 years of network data, predicts audience composition before campaign runs
- **Retail intelligence API** — footfall + mobility data licensed to retailers, property developers, municipalities
- **Journey mapping visualisation** — full city-level audience flow maps

## The Standout Features (USPs)

**Cross-site audience journey** — frequency capping for physical advertising. Agencies have waited 20 years for this. Verified unique reach across a multi-site campaign.

**OmniCollect Score** — once it exists, everyone references it. Like a hotel star rating. Becomes the industry standard.

**Creative performance scoring** — which billboard creative actually works in the real world. Not a focus group. Real data.

**Post-campaign report** — writes itself in 30 seconds with verified figures. Replaces the manual report agencies spend hours on.
