# OmniCollect Business Model

*Locked 2026-06-23*

## Four Customer Types

### 1. Media Owners (primary)
Pay because verified data = higher rates. A site charging R 8 000/month with estimated data can charge R 12 000–R 15 000 with OmniCollect verification. Subscription pays for itself in the first upgraded contract.

| Plan | Sites | Price |
|---|---|---|
| Starter | 1–10 | R 800/site/month |
| Growth | 11–50 | R 600/site/month |
| Network | 51–200 | R 450/site/month |
| Enterprise | 200+ | Custom |

### 2. Agencies
Pay for verified campaign reports and audience planning tools.

| Product | Price |
|---|---|
| Campaign Verification Report | R 4 500/campaign |
| Agency Planning Dashboard | R 8 500/month |
| Custom Audience Segment Report | R 12 000/report |

**Key play:** One major agency mandates "OmniCollect-verified sites only" → every media owner scrambles to join. This is the network tipping point.

### 3. Data Licensing (adjacent industries, pure margin)

| Buyer | Price |
|---|---|
| Retailers (site selection) | R 25k–R 80k/report |
| Property developers | R 35k/site assessment |
| Municipalities | R 50k–R 200k/city contract |
| Insurers | R 80k–R 300k/annual licence |
| Investment funds | R 150k–R 500k/annual licence |

Kicks in at 100+ sites in a geographic cluster.

### 4. Programmatic OOH API
Revenue share: 5–10% of every programmatic transaction using OmniCollect data (Hivestack, Vistar Media, Broadsign). Scales globally without additional hardware.

## The Bundle (where serious money is)

Deckly (R 1 500/mo) + OmniCollect (R 600/site × 20 sites) = **R 13 500/month per media owner**

50 media owners = **R 8M ARR** without a single enterprise deal. Rate cards auto-populated with verified real audience data — product sells itself.
